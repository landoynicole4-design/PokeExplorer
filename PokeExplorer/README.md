# PokeExplorer

## How to Run

1. Open this folder in VS Code
2. Open terminal (Ctrl + `)
3. Run:

npm install --legacy-peer-deps

npx expo start --clear

4. Scan QR code with Expo Go on your phone

## Features
- Browse first 150 Pokemon
- Search by name
- Filter by type (Fire, Water, Grass, etc.)
- View detailed stats, abilities, moves
- Pull to refresh
- Error handling + loading states

## API Used
PokeAPI - https://pokeapi.co (free, no API key needed!)
